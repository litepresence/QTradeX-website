# Optimizers API

All optimizers live in `qtradex.optimizers`.

| Optimizer | Description |
|-----------|-------------|
| QPSO | Quantum-behaved Particle Swarm Optimization |
| LSGA | Linear Selection Genetic Algorithm |
| IPSE | Iterative Parameter Sweep Engine |
| AION | AI-driven neural optimizer |
| MouseWheelTuner | Interactive GUI-based manual tuning |

IPC/candle cache lives in `qtradex/common/pipe/` (JSON file-based).
Tune files are cached per-bot in `tunes/` next to the bot source.
